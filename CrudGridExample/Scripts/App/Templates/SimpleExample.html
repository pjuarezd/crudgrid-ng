How to obtain a basic Grid

First we'll need to setup the Angular controller, for example see SimpleExampleController.js:
Please note the declared variable $scope.gridOptions, which is where we will make happen most of the magic of the grid.
Inside of this variable we'll declare the field types, callbacks, events, and so on... for now it's enough to let it as empy object. 

app.controller('CustomExampleController', ['$scope',
    function ($scope) {

        init = function () {
            $scope.gridOptions = {};

        };
        init();
    }]);

Afther that in the markup we needed render the grid indicating the attribute crud-grid, the table-name attribute, which is where the grid will compsum the REST services and the variable name of the grid options (in this case, called $scope.gridOptions: 

<div crud-grid table-name='Person' options="gridOptions"></div>
